# Digital Compass — Vizyon Belgesi

## Ne ve Neden

Google Play Store'da yükselen pusula app'lerinden biri (`com.compass.digital.direction.directionfinder.pro`) kalitesiz ama hızla yükseliyor. Kullanıcılar reklam bombardımanından, yanlış yön göstermeden ve gereksiz izinlerden şikayetçi.

**Amacımız:** Basit, reklamsız, doğru çalışan bir pusula app'i ile bu boşluğu doldurmak.

## Kim İçin

- Outdoor kullanıcılar (yürüyüşçüler, kampçılar)
- Seyahat edenler
- Basit, güvenilir bir pusula arayan herkes

## MVP Özellikleri (Kapsam İçi)

- [x] Pusula (SensorManager: accelerometer + magnetometer)
- [x] Derece gösterimi (0-360°)
- [x] Yön adı (N, NE, E, SE, S, SW, W, NW)
- [x] Dark theme, minimal UI
- [x] Portrait lock
- [x] ZERO reklam
- [x] ZERO gereksiz izin

## Kapsam Dışı (Sonra)

- [ ] Altitude / basınç
- [ ] Widget
- [ ] AdMob (opsiyonel, kullanıcı tercihli)
