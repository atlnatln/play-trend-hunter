# Çini Boyama — Vizyon Belgesi

## Ne ve Neden

Kültürel mirası dijital boyama terapisiyle birleştiren bir Android uygulaması. Kullanıcılar geleneksel İznik ve Osmanlı çini desenlerini interaktif olarak boyayarak rahatlayacak, aynı zamanda çini sanatıyla tanışacak.

## Kim İçin

- Rahatlama/mindfulness arayan yetişkinler
- Çini sanatına ilgi duyan kullanıcılar
- Geometrik desen ve mandala boyamayı sevenler

## MVP Özellikleri (Kapsam İçi)

- [x] 4 adet hazır çini deseni (line art formatında)
- [x] Dokun-boyala (flood fill) mekaniği
- [x] 12 renkli klasik çini paleti
- [x] Sır (glaze) parlaklık efekti toggle'ı
- [x] İlerleme göstergesi (tamamlanan bölgeler)
- [x] Temizle / yeniden başlat
